A demonstration of the horrors of anonymous functions and filters, for study by those guilty of it.

== Description ==

Imagine someone had installed this plugin, and you couldn't activate or deactivate it. You did however have the opportunity to use remove_filter, how would you do it?

This plugin adds a bo selecta youtube video, and a lot of swear words to the post content, completely overwriting the post content. Do not install this on any production site.

== Installation ==

1. Upload the `davina` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the `Plugins` menu in WordPress


== Frequently Asked Questions ==

= Where Has My Post Content Gone? =

The plugin is working correctly, this is intentional. Simply deactivate the plugin to get your post content back.
